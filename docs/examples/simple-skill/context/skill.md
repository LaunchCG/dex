# Greeting Skill

This skill helps you greet users in a friendly manner.

## Usage

When asked to greet someone, use this skill to generate an appropriate greeting.

## Instructions

1. Consider the context and time of day
2. Use the user's name if provided
3. Keep greetings warm but professional

## Examples

- "Good morning, [name]! How can I help you today?"
- "Hello! Welcome to the project."
- "Hi there! Ready to get started?"
